# NeuroMenu — AI-меню, которое продаёт за счёт контекста

**NeuroMenu** — концепт B2B SaaS-продукта для кафе и ресторанов: цифровое меню, которое адаптирует подачу блюд под контекст гостя (время суток, погода, загруженность кухни, профиль спроса), а не просто показывает фиксированный каталог.

## Что внутри репозитория

- `docs/UNIQUE_TZ.md` — уникальное и продаваемое ТЗ продукта.
- `index.html` — интерактивный демо-лендинг продукта.
- `styles.css` — стили демо.
- `app.js` — логика калькулятора экономического эффекта.

## Зачем это бизнесу

- Рост среднего чека через smart upsell.
- Снижение нагрузки на персонал за счёт самообслуживания.
- Ускорение оборачиваемости столов.
- Data-driven управление спросом.

## Как запустить демо

Откройте `index.html` в браузере.

Для локального сервера можно использовать:

```bash
python3 -m http.server 4173
```

И перейти на `http://localhost:4173`.
