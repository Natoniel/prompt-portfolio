# LearnVoyage — учебная экспедиция на Go

Уникальный учебный проект, где обучение представлено как **ежедневная экспедиция**:
вы задаете задачи, отмечаете прогресс и получаете метрику дня — `Momentum Score`.

## Идея проекта

Вместо обычного todo-приложения LearnVoyage фокусируется на **качестве учебного дня**:
- задачи имеют сложность (1..5);
- есть фокус-время (минуты глубокого обучения);
- рассчитывается итоговая энергия дня (0..100).

Это хорошая тренировочная площадка для:
- структур данных и методов в Go;
- валидации входных данных;
- unit-тестов;
- проектирования маленького доменного слоя (`mission`).

## ТЗ (сформировано и реализовано)

1. Создать модель дневного учебного плана (`DayPlan`).
2. Реализовать добавление задач с валидацией заголовка и сложности.
3. Реализовать отметку выполнения задач.
4. Реализовать приоритетный просмотр задач по сложности.
5. Реализовать формулу `Momentum Score`:
   - 70% вес выполнения задач;
   - 30% вес фокус-времени (насыщение на 180 мин).
6. Покрыть ключевую логику тестами.
7. Сделать консольный пример запуска.
